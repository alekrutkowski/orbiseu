# orbiseu 0.1.0

* Initial package translating the Kalemli-Özcan et al. (2024) Orbis representativeness workflow into R.
* Added firm-year panel construction from relative-year BvD exports.
* Added historical vintage merge helpers.
* Added account duplicate and switcher handling.
* Added ownership link parsing and foreign ownership aggregation.
* Added Eurostat access wrappers using `eurodata`.
* Added SME share, coverage, and concentration functions.
